🌦️ SkyCast

SkyCast is a modern, lightweight weather web application that provides real-time weather updates for any location worldwide. It delivers clean, fast, and accurate weather insights including temperature, humidity, sunrise/sunset timing, and current conditions using a weather API.

✨ Features
🌍 Search weather by any city or location
🌡️ Real-time temperature updates
💧 Humidity & atmospheric conditions
🌅 Sunrise and sunset timing
⛅ Dynamic weather condition display
⚡ Fast and responsive UI
📱 Mobile-friendly design
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript (or React if applicable)
API: OpenWeatherMap API / Weather API
Styling: CSS / Tailwind / Bootstrap (as used in your project)

⚙️ Installation & Setup

Follow these steps to run SkyCast locally:

# Clone the repository
git clone https://github.com/rutujakolate1703-bot/SkyCast.git

# Move into project directory
cd SkyCast

# Open with VS Code
code .

# Run using Live Server or open index.html in browser
🔑 API Setup

If your project uses OpenWeather API:

Get your API key from https://openweathermap.org/api
Replace API key in your JS file:
const API_KEY = "your_api_key_here";
📌 How It Works
User enters a city name
App sends request to weather API
API returns weather data (JSON)
UI dynamically updates with results
